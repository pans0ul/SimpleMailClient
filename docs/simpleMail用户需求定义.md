# simpleMail-app-用户需求定义
1. 程序可以读取txt中的文件内容（txt的文本内容一般以行分开）。用户可以设定发送的时间和设定发送的条目数量。
2. 程序会展示发送内容的计划表。用户可以看到未来要发送的计划任务
2. cli:程序会打印未来要发送的计划任务
3. 程序当然也可以一次性选择发送，也是一个简易的邮件发送客户端


## p1
1. 程序可以读取txt中的文件内容（txt的文本内容一般以行分开）。用户可以设定发送的时间和设定发送的条目数量。
2. 程序会打印未来要发送的计划任务


## p2
1. 如果发送失败，再进行重新发送，尝试三次，直到成功，如果三次失败后，就跳过
2. 如果程序意外退出，支持续发。（回到最近一次发送成功的任务），并开始后续的计划
3. 发送间隔时间设置可以是以天，小时，分，秒为单位。
4. 程序可以读取txt中的文件内容（txt的文本内容一般以行分开）。用户可以设定每次发送的内容行数，直到文件内容发送完毕。

## p3
1. 支持静态编译，可以独立于其他平台运行
2. 可使用加密，和非加密方式设置登陆账户和密码。

3. 用户可以通过参数传递邮件客户端的相关设置，也可以通过读取文件进行设置。相关设置内容如下：
    ```
        #define SMTP_SERVER "smtp.xxx.com"
        #define SMTP_SERVER_PORT 465

        #define SENDER_NAME "xxxx"
        #define SENDER_EMAIL "xxx@xxx.com"
        #define SENDER_PASSWORD "xxx"

        #define RECIPIENT_NAME "xxx"
        #define RECIPIENT_EMAIL "xxx@xxx.com"
    ```
    如果参数传递与读取文件这两者都没有，就引用头文件的方式，编译到程序里面，保证加密性
